Getting started

  1. Find the instructions that was sent in email here.
  2. Next run the following commands:
install.packages('usethis')
usethis::use_course("bit.ly/emi-combine2019")
  1. If it went well, a new project would have opened from RStudio. If you have an issue here please make sure you flag Emi.
  2. Open the file named install-pkgs.R and run all the commands there. This will install all the packages needed for today’s workshop.
